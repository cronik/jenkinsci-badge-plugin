The Jenkins color name or CSS color of the badge.

The plugin supports colors from the Jenkins palette colors. For example, the following are all valid references to the Jenkins palette colors:

  • addBadge icon: 'symbol-star', text: 'A star', color: 'yellow'
  • addBadge icon: 'symbol-star', text: 'A star', color: 'dark-yellow'
  • addBadge icon: 'symbol-star', text: 'A star', color: 'var(--yellow)'
  • addBadge icon: 'symbol-star', text: 'A star', color: 'jenkins-!-color-yellow'

The plugin supports colors from the Jenkins semantic colors. For example, the following are all valid references to the Jenkins semantic colors:

  • addBadge(icon: 'symbol-star', text: 'A star', color: 'warning')
  • addBadge(icon: 'symbol-star', text: 'A star', color: 'success')

The plugin also supports CSS colors. For example, the following are all valid references to CSS colors:

  • addBadge icon: 'symbol-star', text: 'A star', color: '#7fffd4'
  • addBadge icon: 'symbol-star', text: 'A star', color: 'rgb(0, 0, 256)'
